PHASE 4 — BITW MINING ENGINE (CORE SYSTEM)
====================================================

OBJECTIVE:
Build DAILY LOGIN MINING SYSTEM.

CORE MECHANIC:
User must log in daily to activate stone mining.

RULES:
- Login = triggers mining
- No login = no reward for that day
- Missed day = lost yield permanently
- Missed day still counts in duration

YIELD FORMULA:
daily_reward = capital × (yield_rate / 100)

TRACKING FIELDS:
- current_day
- last_login_date
- missed_days
- total_earned
- status

LOGIN FLOW:
1. Authenticate user
2. Fetch active BitW stones (plans)
3. Check last login date
4. Apply mining logic
5. Update wallet
6. Update progress tracker

OUTPUT:
✔ Fully functional BitW mining engine
✔ Daily login-based earning system
